Detailed Description

Utilities for working with the Hough Transform.

Author
Matthijs Bakker This code unit provides various utilities for working with the Hough Transform, including a function to generate the Hough Space for an image, and tools for extracting/ plotting lines resulting from this space.

Use the Hough Transform to isolate lines.

Isolate lines within an image by using Classical Hough Transform.

This function returns the accumulator and the resulting lines.

Parameters
srcThe input image, which data will be read
spaceThe resulting accumulator / Hough space
rnormalsOutput for the normals array
minMinimum value of theta to compute rho for
maxMaximum value of theta to compute rho for
thresThreshold for accumulator values
Returns
Zero or higher, indicating the amount of lines that were detected

Resolve a line from polar coordinates to Cartesian coordinates.

Resolve a line from it's r,t value within Hough parameter space to its coordinates in Cartesian space.

Note that the start and end values will also be tried to match.

Parameters
imageThe image with the sizes of the Cartesian space
spaceThe Hough space
lineThe line to resolve
Returns
A resolved line structure with the {start,end}-{x,y}
