Detailed Description

Utilities for working with the Hough Transform.

Author
Matthijs Bakker This code unit provides various utilities for working with the Hough Transform, including a function to generate the Hough Space for an image, and tools for extracting/ plotting lines resulting from this space.

◆ lane_hough_normal_t

typedef struct normal lane_hough_normal_t

A line represented by rho, theta values.

A normal line which is represented by:

rho = x * cos(th) + y * sin(th)

where rho is the distance from the origin and th is the angle between the line and the x axis.

Resolve a line from polar coordinates to Cartesian coordinates.

Resolve a line from it's r,t value within Hough parameter space to its coordinates in Cartesian space.

Note that the start and end values will also be tried to match.
